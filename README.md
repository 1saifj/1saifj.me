# vm.codes

Personal website &amp; blog.

## Features

- [x] Built with **[Next.js]** + **TypeScript**
- [x] Syntax highlighting with **[Shiki.js]** + [custom theme].
- [x] Markdown with YAML frontmatter.
- [x] **[Open Graph]** support.
